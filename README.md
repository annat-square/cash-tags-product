# Cash Tags — Product Resources

Product documentation and requirements for physical CashTags (Tags).

## What are Tags?

Tags are physical NFC-enabled payment devices (wands, mini cards, etc.) that customers can order through the Cash App. They are powered by the Taply integration and surfaced through the Whimsicard service.
